# Ice Empires — Visual & Art Direction

**Date:** 2026-06-27  
**Purpose:** Guide look/feel while preventing visual polish from distracting from gameplay validation.

---

## 1. Visual north star

Ice Empires should look like:

> A premium indie strategy game about building a hockey empire in a mythic frozen world.

Not:

- Fantasy hockey website
- Enterprise sports dashboard
- Spreadsheet sim
- Cartoon mobile game for kids
- NHL clone
- Generic sci-fi hex board
- Catan clone

---

## 2. Core visual pillars

### Frozen empire

Keywords:

- Ice
- Fog
- Tundra
- Dark water
- Floodlights
- Rinks
- Banners
- Steel
- Frosted glass
- Ancient map / modern arena hybrid
- Dynasty

### Hockey civilization, not NHL skin

Use:

- Outdoor rinks
- Frozen ponds
- Club crests
- Scouting folders
- Sticks, tape, skates
- Old barns
- New development centers
- Map pins
- Prospect cards
- Rival GM portraits

Avoid:

- Real NHL logos
- Sportsbook UI
- Broadcast package look
- Branding too close to existing teams

### Board-game clarity

Use:

- Cards
- Panels
- Icons
- Progress bars
- Clear hierarchy
- Strong hover/click states

Avoid:

- Dense stat tables
- Low-contrast decorative UI
- Tiny all-caps everywhere

### Mythic map

The map should be a randomized hockey world, not literal Earth and not abstract space.

Desired:

- Earth-like landmasses
- Coastlines
- Frozen lakes
- Tundra
- Forests
- Mountains
- Urban regions
- Desert/dryland regions
- Fog of war
- Hockey overlays revealed after exploration

---

## 3. Color direction

Base palette:

- Deep navy / near black
- Blue-black ice
- Slate
- Frosted blue-gray
- Cold white
- Muted steel

Arizona Monsoon accents:

- Monsoon teal/cyan
- Electric storm blue
- Warm desert orange
- Sand/gold highlights
- Occasional lightning yellow

Resource hints:

- Budget: gold/warm yellow
- Operations: steel/gray/orange
- Hockey Knowledge: cyan/blue
- Reputation: purple/white-gold

---

## 4. Typography

Direction:

- Strong readable sans-serif
- Slightly condensed strategy-game feel
- Uppercase labels sparingly
- Big numbers for resources
- Comfortable flavor text

Avoid:

- Overly futuristic fonts
- Comic fonts
- Tiny labels everywhere

---

## 5. UI layout principles

### Top bar

Always visible:

- Club name
- Era
- Month/turn
- Budget
- Operations
- Hockey Knowledge
- Reputation

### Main area

For First 12 Months: dashboard/card flow is acceptable.

For map prototype: map dominates.

### Right panel

Context-sensitive details for club, region, unit, prospect, rival GM, or event.

### Bottom action bar

Current action buttons: End Month, Found Club, Select Build, Select Research, Scout Region.

### Event log

Readable chronicle, not debug console.

## 6. Arizona Monsoon HC identity

Feel:

- Desert storm
- Unlikely hockey market
- Heat meeting ice
- Sudden momentum
- Expansion underdog

Logo placeholder ideas:

- Monogram “AM”
- Storm cloud over rink
- Lightning bolt crossing hockey stick
- Desert mountain with ice-blue storm
- Circular badge/crest

Colors:

- Deep navy base
- Monsoon teal
- Desert orange
- White/ice trim

Copy:

> A storm is gathering in the desert.

---

## 7. Rival GM leader screens

This is a major differentiator.

Should feel like:

- Civ leader screen
- Hockey front office personality
- Negotiation table
- Slightly theatrical
- Static art now, animated energy later

Each GM screen should include:

- Portrait placeholder
- Club crest
- Relationship meter
- Personality tags
- Dialogue
- Response buttons

Visual archetypes:

- Mikko Vaaranen — cold northern tactical severity
- Rex Malloy — old-school prairie barn intimidation
- Elena Novak — elegant diplomatic sharpness
- Carter Briggs — polished territorial confidence
- Valentina Sokolov — dark mysterious patience

---

## 8. Card art direction

### Prospect cards

Feel like scouting reports with card-game clarity.

Include:

- Position badge
- Age
- Potential range
- Scouting Confidence
- Risk
- Traits
- Flavor report

### Staff cards

Include:

- Role
- Specialty
- Monthly effect
- Personality/flavor

### Region cards

Include:

- Terrain
- Region name
- Hockey resource
- Scout report
- Discovery state

---

## 9. Illustration style

Good directions:

- Semi-realistic painted strategy portraits
- Clean editorial sports illustration
- Board-game concept art
- Dark atmospheric map art
- Stylized but not childish

Avoid:

- Cartoon kids-game look
- Hyper-realistic NHL broadcast look
- AI-art weirdness as final style
- Overly busy fantasy illustration

---

## 10. Animation direction

Early:

- Resource ticks
- Card reveals
- Month transition
- Fog reveal
- Rival GM popup entrance

Later:

- Rival GM idle animation
- Weather/fog movement
- Tournament result animation
- Draft reveal tension

Do not start with advanced animation.

---

## 11. Visual prompt examples

### General art direction

```text
Create a premium indie strategy-game visual direction for Ice Empires, a turn-based 4X hockey civilization game. The style should combine Civilization-like map strategy, hockey club identity, fog-covered frozen terrain, and modern board-game clarity. Avoid NHL branding, cartoon kids-game style, fantasy hockey website aesthetics, and spreadsheet-heavy sports dashboards.
```

### Arizona Monsoon crest

```text
Design a fictional hockey club crest for Arizona Monsoon HC. The crest should combine desert storm imagery, ice hockey identity, monsoon teal, deep navy, and desert orange. It should feel like a premium fictional club in a strategy game, not an NHL logo clone.
```

### Rival GM portrait

```text
Create a Civ-style rival GM portrait for a fictional hockey strategy game. The character should feel like a dramatic hockey executive leader screen, with a club crest, moody lighting, and negotiation energy. The style should be semi-realistic premium strategy-game illustration, not cartoonish.
```

---

## 12. Anti-distraction rules

1. Do not polish what is not playable.
2. Timebox visual passes.
3. Do not chase final branding too early.
4. Use visuals to test the fantasy.
5. Avoid visual debt that will be hard to implement later.

---

## 13. Visual milestone order

1. Dashboard readability
2. Arizona Monsoon identity
3. Region/prospect card style
4. Month transition/event polish
5. Map terrain/fog style
6. Rival GM leader-screen style
7. Full art system

Do not start with #7.
